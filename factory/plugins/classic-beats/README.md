# Classic Beats

A four-voice **sample-based drum machine** in the spirit of the early-1980s
sampled rhythm boxes. Four percussion one-shots — **kick, snare, closed hi-hat
and clap** — are decoded from base64 Int16 PCM that is baked directly into the
WASM module, so the plugin is fully self-contained (no runtime file access).

Incoming MIDI notes are mapped to a voice by **note number modulo four**, so
every key triggers a hit:

| note % 4 | voice |
|----------|-------|
| 0        | Kick  |
| 1        | Snare |
| 2        | Closed Hi-Hat |
| 3        | Clap  |

Voices are polyphonic one-shots (a 12-slot pool), so overlapping hits sum
naturally.

## Parameters

| # | Name  | Description |
|---|-------|-------------|
| 0 | Level | Master output level |
| 1 | Kick  | Kick voice level |
| 2 | Snare | Snare voice level |
| 3 | Hat   | Closed hi-hat voice level |
| 4 | Clap  | Clap voice level |
| 5 | Tone  | High-pass brightness applied to hat + clap |
| 6 | Decay | Hit length / amplitude fade-out |
| 7 | Drive | Analog-style soft-clip saturation |

## Build notes

- `assembly.ts` was generated by `classic-beats-gen.mjs`, which embeds the four
  `.b64` PCM blobs and the base64 decoder lifted from
  `factory/tools/SAMPLE_TEMPLATE.ts`.
- Total embedded audio ≈ 87 KB of base64 (kick 17 KB, snare 32 KB, hat 13 KB,
  clap 24 KB), ~1.49 s of audio at 22050 Hz mono.
- Verified with `factory/tools/wasm-runner.mjs … --synth` → **VERDICT: PASS**.
  (The runner's built-in parameter sweep plays a single note, which maps to one
  voice; a four-note check confirms all eight parameters audibly affect the
  output.)
