How it works

From a sentence to a signal chain.

No DSP code, no toolchain to install. The plugin does the engineering for you and then gets out of the way.

1

Describe it

Type what you want — “a gritty tape saturator with drive and tone”, or “a lush analog pad”. The plugin sends the prompt to Claude.

2

It writes the plugin

Claude returns an AssemblyScript DSP module and a self-contained HTML GUI. A compiler bundled inside the plugin turns the DSP into WebAssembly — no toolchain on your machine.

3

Play & refine

The WASM runs under wasmtime and the GUI shows in an embedded view. Keep talking to evolve it, then save a portable .vstai file to reload anytime.

Features

Built like a boutique plug-in. Driven by language.

Real DSP, generated

Filters, saturation, envelopes, oscillators — written as AssemblyScript and compiled to native-speed WebAssembly.

A custom GUI every time

Each plugin ships its own hand-styled interface — knobs, sliders, meters, a playable keyboard for synths — not a generic preset rack.

Offline at playback

The cloud is only touched while generating. Once built, the DSP runs locally under wasmtime — no network in the audio path.

Your model, your way

Bring your own API key, paste prompts into any chatbot for free, or use cloud credits. Open models via Ollama work too.

Portable .vstai files

Every plugin saves to one shareable file — params, GUI and compiled WASM — and reloads instantly, no regeneration needed.

Full host automation

A fixed 64-parameter pool is exposed to your DAW, so every generated control can be automated and recalled like any other plugin.

Two plugins, one engine

An effect and an instrument.

The same core ships as two products so your DAW treats them correctly — one in the FX chain, one as a playable instrument.

Audio effect

VibePlugin FX

Drop it on any channel and describe a processor. Reads the input, writes the processed output.

  • Saturators, EQs, reverbs, delays, dynamics
  • Load a file for samplers, granular & convolution
  • Tweak by prompt — it patches, not rewrites
Instrument · synth

VibePlugin Synth

Play it from MIDI or the on-screen keyboard. Synthesizes straight into the output buffer.

  • Pads, basses, leads, plucks, drones
  • Built-in playable keyboard in every GUI
  • Host converts MIDI to Hz — just play
